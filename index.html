📄 GitHub Pages — Instrucciones

    

      

        
1. Creá un repositorio GitHub (público) o usá uno existente.

        
2. Subí este archivo HTML como index.html en la raíz del repo.

        
3. Activá GitHub Pages: Settings → Pages → Branch: main → Save.

        
4. Tu app queda en: https://TU-USUARIO.github.io/TU-REPO/

        
5. Abrí la app en el browser, andá al tab Código GAS y configurá la URL del WebApp.

        
6. Las llamadas al Sheet se hacen directamente desde el browser → GAS → Google Sheets.

      

      

        Nota sobre CORS: El GAS WebApp acepta peticiones desde cualquier origen porque usamos
        Content-Type: text/plain (evita el preflight OPTIONS).
        El header Access-Control-Allow-Origin: * está configurado en el Código.gs.
